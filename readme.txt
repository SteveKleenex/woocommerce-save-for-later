=== WooCommerce: Save For Later ===
Contributors: codearachnid 
Tags: woocommerce,wishlist,account,ecommerce
Requires at least: 3.4
Tested up to: 3.5
Stable tag: trunk

Allow your visitors/customers to add products to a personal wishlist that they may save to purchase later or share with their friends.


== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
